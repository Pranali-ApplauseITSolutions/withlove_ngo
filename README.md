# PrayasYouthForum

This project is a web application for the Prayas Youth Forum, a non-profit organization focused on youth empowerment and social initiatives in India.

## Features

- Informational pages about the organization, its mission, vision, and team
- Event and news management (add, edit, delete events and reports)
- Donation information and contact details
- Volunteer registration
- Blog and gallery sections
- Admin dashboard for managing content
- Privacy Policy and Terms & Conditions pages

## Technologies Used

- HTML, CSS (Bootstrap, custom styles)
- PHP (for dynamic pages and admin functionality)
- JavaScript (jQuery, custom scripts)
- MySQL (database, see [`config.php`](PrayasYouthForum/config.php))
- [PHPMailer](PrayasYouthForum/PHPMailer/README.md) for email functionality

## Getting Started

1. **Clone or Download** the repository.
2. **Set up a local server** (e.g., XAMPP) and place the `PrayasYouthForum` folder in your web root.
3. **Configure the database** in [`config.php`](PrayasYouthForum/config.php).
4. **Import the database schema** (not included here; create tables for users, events, reports, etc.).
5. **Access the site** via `http://localhost/PrayasYouthForum/`.

## Customization

- Update organization details in the relevant HTML files.
- Adjust styles in [`css/main.css`](PrayasYouthForum/css/main.css).
- Add or modify PHP scripts for backend logic as needed.
